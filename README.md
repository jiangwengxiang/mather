### 简介
这是一个面向学生的公共项目，旨在帮助他们练习数学能力，并将其作为数学工具使用。    
This is a public programme designed for students to practice their maths capability  and use as a maths tool .
### 功能
1. 数学练习,例如可以进行加减乘除的运算训练。
2. 数学工具,例如可以进行图形面积计算，体积计算等。
3. 数学游戏（待开发）
